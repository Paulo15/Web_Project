Projeto consiste em criar um sistema de ponto eletrônico, onde usuário lança sua jornada, licensas médicas, ausência, e etc.
Gestores devem realizar as aprovações e eventuais consultas aos reigstros podem ser feitas tanto por gestores, quanto para funcionários.

Layout(Telas):

- Tela Login
- Tela Menu
	- Funcionalidades:
		- Lançamento de horas
		- Consulta de Jornada
		- Registro de ocorrências


Requisitos:
- 2 ou mais perfis de usuário
- Armazenamentos informações em Banco de dados SQL
	- Registro de jornadas
	- Registro de férias
	- Registro de licensa médica
	- Registro de funcionários
	- Registro de gestores

Tarefas:

Implementação tela login (TO DO)

Implementação reconhecimento de usuário e senha (TO DO)

Implementação cadastro de usuário (TO DO)

Implementação visão relatório da jornada (TO DO)

Implementação Procedures para inserir informações (TO DO)

Implementação tela inicial (TO DO)
